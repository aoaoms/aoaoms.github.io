                    # 关于微胖viewUpon

                    做了几个微信小程序、小游戏小项目，使用了微信云开发（免费的云资源）；后期维护数据发现比较麻烦，需要开发工具登录每个小程序进入云开发管理，于是就有了想做一个一站管理小程序云开发资源（数据）的系统。优点如下：

                    1. 多个小程序云资源一站管理，省去使用官方的繁琐操作。
                    2. 使用官方标准的API接口处理数据。
                    3. 可以自由定制自己的数据展示及处理界面。
                    4. 更多可扩展的数据处理及分析功能。

                    # 登录\注册\重置密码

                    如您看到此文档，则您可能已打开微胖 登录界面；没错，由于使用自有的服务器资源；为了避免接口滥用以及尽可能好的管理资源请求，项目做了注册、登录、重置密码等基础功能。没有账号你可以先注册一个；经过注册信息验证后，你会收到一封含有验证码的邮件，请按照提示进行注册。注册成功后，你就可以登录系统使用微胖啦。(^ ^)。 同样，如果你注册了 但忘了密码时，你可以重置密码；重置密码需要你输入自己的注册邮箱，验证通过后你会收到一封验证码操作重置密码。

                    PS: 目前不限定注册用户数量，后续不排除限制注册、限制使用API资源等操作。

                    # 环境

                    环境对应于你的一个小程序使用的云开发资源环境；微信小程序云开发赋能用户可针对一个小程序创建2个云环境，你可以使用免费的云资源，具体可参阅微信小程序云开发;对于很多小的小程序项目，个人觉得此免费的云资源够用。 在微胖的环境菜单页，你可以操作新增环境、编辑环境、刷新访问凭证token(此凭证用于你访问小程序云api)、移除环境、查看环境集合；并可在右侧集合界面操作集合！ 新增环境需要你添加小程序相关的信息：

                    1. 小程序appid
                    2. 小程序secret
                    3. 其他辅助信息

                    在微胖里面这些信息都仅保存在localStorage里面。 新增环境基本信息后，你需要操作【刷新token】；此操作会同步获取最新的小程序云API访问凭证，此凭证是全局的。意思为，如果你的小程序业务逻辑中有其他地方也使用了此token，并进行了缓存；那么这些逻辑即时处理时可能会报错，不是最新的token！更多解释可查阅小程序API凭证。 另外，此token有效期限定2小时。

                    为什么要这样设置？ 这些重要数据，微胖不做管理；仅在使用api的时候需要提交请求处理。你需要明确知晓并保管好本地数据。

                    # 新增环境

                    环境页面，点击新增按钮，添加一个微信小程序环境。

                    1. 小程序名称：你的小程序名称
                    2. 小程序appid: 你的小程序appid
                    3. 小程序密匙：你的小程序密匙secret
                    4. 云环境ID: 你的小程序云环境id
                    5. 云环境别名：自定义环境别名
                    6. 小程序全局调用token: 无法编辑 由接口调用刷新
                    7. 腾讯云调用token： 无法编辑 暂无作用 后续由接口调用刷新 进一步操作腾讯云api
                    8. 备注：自定义环境说明

                    # 编辑环境

                    环境操作，点击编辑，修改本地环境配置信息

                    属性同上

                    # 刷新token

                    环境操作，点击刷新token，调用接口获取最新小程序云api调用凭证access_token。刷新后保存在本地。

                    # 移除

                    环境操作，点击移除，删除本地存储中的当前环境信息。

                    # 浏览集合

                    环境操作，点击集合，打开右侧集合信息浏览界面。操作当前环境内的集合信息。可新增集合、编辑集合别名、删除集合、同步集合。

                    # 集合操作

                    1. 新增集合 ：当前环境下新增一个集合
                    2. 同步集合 ：同步当前环境下的所有集合到本地
                    3. 浏览集合记录 ：进入集合-记录页面，可进一步操作集合记录；查询、编辑、删除
                    4. 集合别名定义 ： 自定义集合别名
                    5. 删除集合 ： 删除当前环境下的当前集合

                    # 集合

                    菜单导航进入集合操作页面；选择云环境、集合后，可操作相应集合数据。新增记录、查询记录、编辑记录、删除记录。

                    1. 下拉选择已录入的环境
                    2. 下拉选择选择的环境内的集合
                    3. 按钮-当前集合里新增记录
                    4. 按钮-刷新当前集合页面数据
                    5. 精确查询- 下拉选择字段 （如果你不知道要查的字段名，可以先刷新一下，会自动加载当前集合的字段名到下拉列表）
                    6. 精确查询- 输入要查询的字段的值
                    7. 按钮- 查询 执行精确查询
                    8. 按钮- 重置精确查询条件
                    9. 数据展示- 当前活动的集合数据
                    10. 分页信息

                    # 新增记录

                    注意！新增记录前，强烈建议你先刷新一下集合信息；这样的话，新增时会默认加载一个数据模板 提供你新增编辑记录使用。

                    系统使用的对象字段查看、新增记录、编辑记录操作均使用的 vue-json-editor

                    # 精确查询

                    注意！使用精确查询前，强烈建议你先刷新一下集合信息；这样的话，集合字段名会默认加载到查询字段选择下拉列表，提供你选择要查询的字段。 录入完查询字段后，输入要查询的字段值，点击【查询】即可精确查询集合记录。

                    # 重置查询

                    按钮- 重置查询条件

                    # 对象字段查看

                    由于二维表展示集合记录，有些字段可能定义为数组或对象；这时，展现数据时会提示为对象字段；点击 对象字段则会弹窗显示详细数据内容，如图。

                    # 编辑记录

                    数据行，点击【编辑】可对当前行记录进行编辑。

                    注意！编辑数据 更新操作默认是使用微信小程序云环境记录默认字段【_id】。当前记录若无_id 字段，系统会提示无法【提交】操作。

                    # 删除记录

                    数据行，点击【删除】可删除当前记录。

                    注意！删除数据操作默认是使用微信小程序云环境记录默认字段【_id】。当前记录若无_id 字段，系统会提示无法【提交】操作。

                    删除数据，需谨慎！

                    # 错误处理


                    当返回access_token 过期时，请在环境页面，刷新相应环境的token 即可。因每个token的有效期仅2小时，多数情况下你登录系统操作数据前就需要刷新一下本地token。

                    其他接口错误会返回相应的错误码，具体原因可参考微信官方接口错误说明错误码
